# 🧮 Calculator App

A simple and modern Calculator built using HTML, CSS, and JavaScript.

This project performs basic arithmetic operations with a clean UI and smooth button animations. It is beginner-friendly and demonstrates core JavaScript concepts like DOM manipulation and functions.

---

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear button (C)
- Delete last character (⌫)
- Responsive layout
- Modern gradient UI with hover effects

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (DOM Manipulation)

---

## 📂 Project Structure

Calculator/
│
├── index.html
├── style.css
└── script.js

---

## 💻 How It Works

- When a button is clicked, its value is appended to the display.
- The calculate function evaluates the expression.
- try-catch is used to prevent crashes from invalid expressions.
- CSS enhances the UI with animations and gradient backgrounds.

---

## 🔧 How to Run the Project

1. Download or clone the repository
2. Open index.html in your browser
3. Start performing calculations
